Create a to-do list that is simple to use and beautiful to look at.

  • Develop web application using AngularJS to simplify development of page updates without constant refreshes.
  • Implement real-time push updates to all devices using NodeJS and Socket.IO.
  • Full drag-and-drop interface, everything is point and click, no page reloads.
  • Repeating tasks allow users to automatically enter a task across a range of days.
  • Ability to share tasks with friends or collegues.
  • User selectable custom themes to allow changes to images, colours and fonts.
