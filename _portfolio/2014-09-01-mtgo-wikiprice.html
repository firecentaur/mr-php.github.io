Develop a website to display MTGO card prices from a collection of clients.

  • Import and sync over 9 million rows hourly with CSV data loaded by over 1000 clients.
  • Determine best buyer/seller in realtime based on current data.
  • Display data, allowing the user to filter and order the results using advanced joins and sorting.
  • Obfuscate prices as images to prevent page scraping.
  • Create a stunning visual experience for the user.
