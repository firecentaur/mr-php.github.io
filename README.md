# Mr PHP Website

Yep, not a single line of PHP code to run MrPHP.com.au, how ironic.

## Notes and How-To's

If you're editing this site then you will probably want to get familiar with [jekyll](http://jekyllrb.com/docs/).

### Redirects

Add to page front matter:

```
---
redirect_from:
- "/blog/some-old-slug/"
---
```
